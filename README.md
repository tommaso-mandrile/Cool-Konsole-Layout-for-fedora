## To Do before launching COOLKonsoleStarter.sh 
Open the terminal in this folder and run installPrograms.sh

---

##### **installPrograms.sh** will:
+  Copy "BashCommands/fire.sh" in "usr/local/bin" under the name of "fire_effect"
+ Install requested applications (btop, fastfetch,cacafire)


> [!NOTE]
>     I heavely suggest to assign a shortcut (like "Meta+Enter")
>     to run the COOLKonsoleStarter.sh program so that you can
> 	  open this layout whenever you want.
> 	  **To do this:** go to System Settings --> Keyboard --> Shortcuts --> Add New -->
> 	  Command or Script --> Choose...
> 	  Then choose the path that leads to COOLKonsoleStarter.sh and assign a shortcut.
